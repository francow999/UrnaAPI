# Urna API
Este é um projeto que comecei o desenvolvimento em 2026
com o intuito de ajudar meu irmão em um problema
real de uma escola. No programa, você deve escolher dentre os 5
professores que você mais tem interesse. Como resultado, será feito uma tabela
para cada tipo (tipo 1, tipo 2, ...).

# Requisitos e Ferramentas Utilizadas
- Node.js
- PostgreSQL
- ExpressJS
- Git
